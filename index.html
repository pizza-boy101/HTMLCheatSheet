My Quick HTML Cheat Sheet!

All of these elements have closing tags unless specified to not have, example: <h2> has to end with </h2>

Important HTML Elements
<head> A place to store meta data and
add custom fonts and connect
to css files and etc.
<body> Where you can enter elements
for text to display
<div> A divider element where
you can assign ids and classes to it
<h1-6> A header tag with 6 different sizes,
1 being the biggest 6 smallest.
<p> p stands for paragraph and
the default font size is 16px

Visual Sub Elements
<strong> Bolds the text that are in-between
the tags
<em> Turns the text in the closing element to italics
